
# ToDo

Essa é uma aplicação de lembrete de tarefas, desenvolvida para treinar conceitos básicos de React Native.  


## Funcionalidades

- Contagem de tarefas;
- Adicionar uma nova tarefa;
- Remover uma tarefa;
- Marcar uma tarefa como concluída;
- Desmarcar uma tarefa como concluída;


## Instalação

Execute no ExpoGo:

```bash
  npm install
  expo start
```

Execute no Android:
```bash
  npm install
  expo run:android
```

Execute no IOS:
```bash
  npm install
  expo run:ios
```
